## docker

Para rodar usando docker:

1.  cd Catalog
2.  `docker run -d -p 5432:5432 --name csd-crudb -v postgres:/var/lib/postgresql/data postgres:9.5`
3.  `docker build -t csd-crud .`
4.  ``docker run -p 3000:3000 --link db:db --name app_container app```
